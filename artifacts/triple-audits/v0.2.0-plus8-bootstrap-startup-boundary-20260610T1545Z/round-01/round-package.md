## Frozen Audit Goal

Audit whether the current local implementation:

1. establishes one canonical tenant-public identity-readiness boundary;
2. removes broad auth bootstrap side effects from protected public request helpers;
3. fixes the first permission-granted web map entry by cause-root ownership rather than mutable runtime hacks;
4. preserves the absorbed Home-start rule that anonymous tenant Home stays on the public surface unless there is a real invite/deferred override;
5. avoids gambiarra / hidden compatibility shims in the final architecture.

## Implementation Intent

- Protected tenant-public request helpers must no longer call broad `AuthRepository.init()` just to obtain an anonymous bearer.
- Web location permission grant must not leave the permission screen until the runtime has a canonical path to finish location bootstrap; if the same SPA document cannot reliably read geolocation immediately after grant, the fix must be document/startup owned instead of a mutable global bypass.
- Anonymous web policy remains `anonymous-authenticated`; raw unauthenticated public reads are not allowed.
